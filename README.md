# Naurok Helper 🎓

Android додаток для допомоги з тестами на Naurok.com.ua

## 🚀 Функціонал

- **WebView браузер** - відкриває Naurok прямо в додатку
- **Gemini AI підказки** - натисни на зелену кнопку 🤖 для отримання підказки
- **Draggable кнопка** - можна перетягувати куди зручно
- **Збереження API ключа** - вводиш один раз при першому запуску

## 📱 Як використовувати

1. **Перший запуск:**
   - Відкрий додаток
   - Введи Gemini API ключ (безкоштовний: https://aistudio.google.com/apikey)
   - Натисни "Зберегти"

2. **Робота з тестами:**
   - Відкрий будь-який тест на Naurok
   - Натисни зелену кнопку 🤖 внизу справа
   - Додаток витягне питання та варіанти відповідей
   - Gemini AI покаже підказку з правильною відповіддю

3. **Налаштування:**
   - Натисни кнопку підказки → "Налаштування"
   - Можеш змінити API ключ

## 🛠️ Білд через GitHub Actions

1. Завантаж проєкт на GitHub
2. Actions автоматично зібере APK
3. Завантаж готовий APK з Artifacts

## 📦 Технічний стек

- **Kotlin** - мова програмування
- **WebView** - вбудований браузер
- **OkHttp** - HTTP клієнт для Gemini API
- **Material Design** - FAB кнопка
- **Gemini Flash API** - безкоштовна AI від Google

## 🔑 Як отримати Gemini API ключ

1. Йди на https://aistudio.google.com/apikey
2. Натисни "Create API key"
3. Скопіюй ключ
4. Вставляй в додаток при першому запуску

**Безкоштовний ліміт:** 15 запитів/хвилину

## 📝 Вимоги

- Android 10+ (API 29+)
- Інтернет з'єднання
- Gemini API ключ
